Format
Use string replacers with pattern regular expressions to format input
Set attribute format with a string replacer. Currently only the number group selector works ($1, $2, ...). The formatter works off the input pattern regex. You can use groups to split value into parts.
Note if escaping(backslash) in pattern attribute you need to double escape (\s -> \\s)
 
      
  ${escapeHTML(`
  
  
    
    
    

  
  
  
  
  
  
    
    
    

  `)}
      
    
      
  ${`const inputSSN = document.querySelector('input#ssn');
  console.log(inputSSN.value); // the typed input 123121234
  console.log(inputSSN.formattedvalue); // the formatted input 123-12-1234
  
  const inputPhone = document.querySelector('input#ssn');
  console.log(inputPhone.value); // the typed input 1231231234
  console.log(inputPhone.formattedvalue); // the formatted input (123) 123 1234`}
Mask format
Use string replacers with format to mask the input
Set attribute mask with a string replacer. Currently only the number group selector works ($1, $2, ...). The mask requires a format to be set

  console.log(inputSSNMask.maskedValue); // the formatted input xxx-xx-1234
  console.log(inputSSNMask.formattedvalue); // the formatted input 123-12-1234`}
