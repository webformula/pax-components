Material Design Guidelines: Search

Search allows users to enter a keyword or phrase and get relevant information. It’s an alternative to other forms of navigation.

Basic Debounce search Icons History Speech Sections Templates Quick results Filters

Types

Standard
By default the search event only triggers on enter key. Look at next example for how to change that.
      
  ${escapeHTML(``)}
      
    
      
  ${`// search executed (enter key)
  document.querySelector('mdw-search#two').addEventListener('search', event => {
    event.target.updateQuickResults([{
      value: 'one',
      primary: 'One',
    },{
      value: 'two',
      primary: 'Two'
    }].filter(v => v.primary.toLocaleLowerCase().includes(event.target.searchValue)));
  });

  // text input
  document.querySelector('mdw-search#two').addEventListener('input', () => {});

  // search result selected
  document.querySelector('mdw-search#two').addEventListener('change', event => {
    console.log(event.target.value);
  });
  `}
Debounce search on input
Specify the debounce milliseconds in the \`debounce\` attribute. Default is 300. If this is not set then search event is triggered on enter key
You can hide spinner by adding the class name \`mdw-no-spinner\`
        
    ${escapeHTML(``)}
        
      
        
    ${`// search executed (enter key)
    document.querySelector('mdw-search#debounce').addEventListener('search', event => {
      event.target.updateQuickResults([{
        value: 'one',
        primary: 'One',
      },{
        value: 'two',
        primary: 'Two'
      }].filter(v => v.primary.toLocaleLowerCase().includes(event.target.searchValue)));
    });
  
    // text input
    document.querySelector('mdw-search#two').addEventListener('input', () => {});
  
    // search result selected
    document.querySelector('mdw-search#two').addEventListener('change', event => {
      console.log(event.target.value);
    });
    `}

\${data.secondary}

    

  `)}

  document.querySelector('mdw-search#eight').addEventListener('input', event => {
    event.target.updateQuickResults([{

      section: 'quick'
