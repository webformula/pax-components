Theming

Material Theming is a systematic approach to customize your app to better represent your brandMaterial Theming

How does theming work?

There are 3 main colors you can set in material. Primary, secondary, and error. Along side the colors you can adjust the colors that show on them for content like text and icons.There is also 2 contrast setting you can choose. Light (default), and dark (the cool one)

Add you theme
To add a custom theme, use the genroator below and create a css file. Once you have the theme css file you can load that in place of 'theme.css' in pax-components distrobution.
