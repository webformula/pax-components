Material Design Guidelines: Dialog

Dialogs can require an action, communicate information, or help users accomplish a task. There are two types of dialogs: basic and full-screen.

Simple Template Fullscreen Response

Types

Simple dialog
Open 
      
  const answer = await MDWDialog.simple({
    headline: 'Question',
    message: 'Are you sure?',
    actionNegative: true // default false
  });

  if (answer === 'positive') console.log('User pressed ok');
  if (answer === 'negative') console.log('User pressed cancel');
